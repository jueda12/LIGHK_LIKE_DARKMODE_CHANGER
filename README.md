# LIHKG_LIKE_DARKMODE_CHANGER

Chromium extension that toggles a LIHKG-like dark mode palette on any webpage through the right-click context menu.

## Usage

1. Open `chrome://extensions` (or any Chromium-based browser extensions page).
2. Enable **Developer mode**.
3. Click **Load unpacked** and select the extension folder (this repository directory).
4. Open any webpage and right-click.
5. Click **切換 LIHKG Dark Mode 配色** to toggle dark mode on/off.
